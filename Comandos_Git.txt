## Principais comandos Git

-> git clone - Para baixar o projeto a primeira vez na sua máquina;
-> git add . - Para adicionar os arquivos nos commits;
-> git commit -m "Mensagem" - Para deixar uma mensagem de formalização das mudanças;
-> git push - Para fazer o upload das mudanças para o GitHub;
-> git pull - Para fazer o download do código atualizado;
